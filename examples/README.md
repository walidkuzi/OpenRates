# Examples

These examples show how to use OpenRates from an agent, the command line, the REST API, and
the TypeScript SDK.

- `claude-code/` — connect the MCP server to Claude Code.
- `curl.sh` — call the REST API with curl.
- `node/` — use the TypeScript SDK from Node.

All examples use official reference rates from Frankfurter, which need internet access but no
API key.
